# PropSpace - Property Listing App

PropSpace is a full-stack property listing web application where users can register, login, manage their profile, and create, view, update, delete, search, and filter properties for rent or sale.

## Tech Stack

- Frontend: React
- Backend: Node.js and Express
- Database: MongoDB
- Authentication: JWT and bcrypt password hashing

This project is being developed step by step according to the Programming in JavaScript final exam requirements.
